# Image Generation Prompt Pack

Purpose: provide project-bound prompts for producing concept references and eventual source art. Generated images are not final assets until curated, edited, compressed, named with stable content IDs where needed, imported into Godot, and validated on Android.

Default use case: `stylized-concept`.

Global constraints for every prompt:

- No text, no watermark, no logo.
- Elevated 2D/2.5D action-RPG readability.
- Stylized realism, painterly surface detail, practical silhouettes.
- Pleasant but gritty fantasy-western tone.
- Readable on mobile.
- Avoid generic fantasy stock art.
- Avoid dark, low-contrast compositions.
- Avoid friendly horse imagery.

## Production Notes

- Use generated images as references first.
- Convert final art into atlases, spritesheets, or UI textures inside `assets/art/`.
- Create sibling versioned filenames instead of overwriting existing assets.
- Keep transparent sprites on clean alpha only after edge validation.
- Store prompts with asset source notes for traceability.
- Validate every project-referenced image through Godot import and Android smoke tests.
