## Current Prototype Notes

The existing codebase is a wave-based shooter prototype. It proves basic Godot Android wiring and shooting mechanics, but the target product is now:

- continuous open-world RPG,
- authored and procedural content hybrid,
- settlement and follower systems,
- high-detail stylized visuals,
- touch-first Android release.
